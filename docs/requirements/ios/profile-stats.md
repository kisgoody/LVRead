# 我的模块与统计需求（iOS）

## 需求拆解

我的是底部导航第三项，负责展示阅读画像和个人统计。当前阶段不做账号体系，不依赖服务端，所有统计来自本地阅读行为。

## 阅读统计

| 指标 | 口径 |
|---|---|
| 阅读时长 | 阅读页进入前台且非锁屏状态的有效停留时间，连续 60 秒无翻页/滚动/触控则暂停计时 |
| 阅读字数 | 按实际阅读页覆盖的字符区间去重累计 |
| 阅读书籍数 | 统计周期内发生有效阅读行为的 bookId 数量 |
| 连续阅读天数 | 每日有效阅读时长 ≥ 3 分钟计为 1 天 |
| 夜读 | 23:00-06:00 发生有效阅读，且当日累计阅读 ≥ 30 分钟 |

## 统计维度

| 维度 | 展示 |
|---|---|
| 日 | 今日阅读时长、字数、书籍、时间段分布 |
| 月 | 每日柱状趋势、本月总时长、本月总字数、最长连续天数 |
| 年 | 月度趋势、年度总时长、年度总字数、读完书籍数 |

## 总结与建议

- 日总结示例：`今天阅读 42 分钟，主要集中在 22:00 后，建议提前 30 分钟结束阅读。`
- 月总结示例：`本月已阅读 18 天，技术类书籍占比最高，平均每次阅读 21 分钟。`
- 夜读提醒：23:00 后仍在阅读且当日累计超过 30 分钟时，在我的页和阅读页设置入口显示温和提醒。
- 设计总结建议算法
- 阅读建议不强制打断阅读，只提供可关闭的提示。

## 藏书统计

| 指标 | 说明 |
|---|---|
| 总藏书 | 当前书架未删除书籍总数 |
| 阅读中 | 进度 > 0 且 < 100 的书籍 |
| 待读 | 进度 = 0 的书籍 |
| 已读完 | 进度 ≥ 100 的书籍 |
| 格式分布 | TXT、EPUB、PDF、MOBI、AZW3 |
| 来源分布 | 本地导入、分享导入、同网传输 |

## 笔记统计

| 指标 | 说明 |
|---|---|
| 评论数 | 当前未删除评论总数 |
| 书签数 | 当前活动书签总数 |
| 高频书籍 | 评论 + 书签最多的前 3 本 |
| 最近活跃 | 最近一次创建评论或书签的时间 |

## 页面状态

| 状态 | UI 要求 |
|---|---|
| 有数据 | 顶部个人摘要、阅读统计卡、趋势图、藏书统计、笔记统计 |
| 无阅读数据 | 展示统计口径说明和「去书架阅读」入口 |
| 无笔记数据 | 展示从阅读页添加评论/书签的引导 |
| 统计计算失败 | 保留已有缓存统计，提示稍后刷新 |

## 验收标准

- 日/月/年切换不丢失当前筛选状态。
- 阅读时长和字数不因重复打开同一页而重复累计。
- 夜读提醒可关闭，当天不再重复打扰。
- 所有统计卡支持 Dynamic Type 后不重叠。
