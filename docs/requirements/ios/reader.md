# 阅读模块需求（iOS）

## 需求拆解

阅读模块是 LV Read 的核心沉浸场景，负责书籍正文展示、连续页流、翻页/滚动、阅读设置、目录跳转、评论入口、书签入口、阅读进度保存与阅读统计采集。当前阶段仅考虑 iOS，原生实现必须使用 UIKit。

阅读页不作为底部 Tab 展示，从书架进入后全屏打开；返回后回到书架或来源页面。阅读页内部工具栏承载目录、夜间模式、设置、自动阅读、进度跳转、评论和书签状态。

使用iOS 原生翻页控制器 UIPageViewController  实现阅读页面
## 完整代码

### 页面结构

| 区域 | 内容 | iOS 实现要求 |
|---|---|---|
| 正文顶部状态区 | 左侧返回(和章节名称文字高度相同)，右侧章节名称 | 默认显示，不遮挡正文内容，不超出手机安全边界 |
| 正文区域 | 当前页正文、段落、评论标记、书签标识、生肖水印 | 正文不被顶部、底部状态区遮挡，长按段落触发，输入为空禁止保存，点进评论标记可查看修改评论；若章节名称有重复的则去除重复章节，且杜绝出现只有章节无内容的页面 |
| 正文底部状态区 | 左侧章节进度：当前页数/章节总页数，右侧时间和电量 |  默认显示，不遮挡正文内容，不超出手机安全边界 |
| 一级设置菜单页面顶部状态区 | 左侧返回按键，书名居中，右侧分享到PC端按钮 | 默认隐藏，点击屏幕中央，宽度为屏幕宽度的1/5；安全区内布局 |
| 一级设置底部工菜单栏  | 章节目录按钮、夜间模式按钮、主题设置按钮、字体布局设置按钮 | 默认隐藏，点击屏幕中央显示；安全区内布局 |
| 章节目录 Sheet | 章节列表页不超过手机屏幕高度的1/2,默认滚动到当前章节，点进切换章节，手动关闭章节页 | 当前章节高亮，支持超长章节名截断 |
| 夜间模式 | 夜间模式开为纯黑OLED主题，夜间模式关为切换前的主题 | 切换前主题需记忆，关闭夜间模式需恢复切换前的主题 |
| 主题设置 Sheet | 亮度、主题、翻页方式、护眼、字体、字号、行距、段距、页边距、 | 底部动画弹出，动画退出，实时预览 |
| 字体布局设置 Sheet | 字体、字号、行距、段距、左右边距、上下边距 | 底部动画弹出，动画退出实时预览 |

### 书籍加载方案

| 场景 | 需求 |
|---|---|
| 预加载缓冲 | 页面展示时异步预加载前后 2 页文本 / PDF 位图，存入内存缓存池，滑动时直接读取避免主线程阻塞 |
| 资源释放策略 | 超出 ±2 页范围的页面，清空 WKWebView/PDFView、销毁大图、释放纹理内存 |
| 分层渲染 | 文本用 CoreText 后台分页，扫描 PDF 用下采样降低分辨率，滑动结束再渲染高清图 |
| 控制器复用池 | 复用少量 PageVC 实例，避免频繁 alloc 创建销毁对象 |
| 资源释放策略 | 超出 ±2 页范围的页面，清空 WKWebView/PDFView、销毁大图、释放纹理内存 |
| 目录跳转 | 清空当前窗口，以目标章节/页为中心重建窗口 |
| 进度跳转 | 拖动结束后才提交跳转，跳转过程显示骨架页 |
| 首尾边界 | 全书开头/结尾不强制补页，不重复创建页面 |

### 翻页与滚动

| 模式 | 交互 | 实现方式 |
|---|---|---|
| 仿真 | 模拟纸张翻页 | style = .pageCurl，orientation = .horizontal，doubleSided = true |
| 左右翻页 | 平移翻页，左右平移 | style = .scroll，horizontal |
| 上下翻页 | 平移翻页，上下翻页 | style = .scroll，vertical |
| 上下滚屏 | 组合实现长文上下连续滑动阅读 | 通过 navigationOrientation = .vertical + transitionStyle = .scroll 组合实现长文上下连续滑动阅读 |
| 无动画 | 点击或滑动左右区域直接切页 | -- |



### 评论入口

| 项目 | 说明 |
|---|---|
| 触发方式 | 长按当前段落 |
| 选区规则 | 默认选中整个段落，允许调整；最多保存 500 字选中文本 |
| 输入规则 | 评论正文不能为空，最多 1000 字 |
| 保存结果 | 段落侧边显示评论标记；笔记 Tab 同步出现评论卡 |
| 回到原文 | 从笔记页进入时滚动或翻页到对应章节、页码、段落 |
| 异常处理 | 保存失败保留输入内容，并提示「评论保存失败，请重试」 |

### 书签入口

| 项目 | 说明 |
|---|---|
| 触发方式 | 阅读页下拉当前页面 |
| 设置 | 当前页无书签时，下拉后设置书签 |
| 取消 | 当前页已有书签时，再次下拉取消书签 |
| 标识 | 页面顶部或页角显示书签标识，不只依赖颜色 |
| 去重 | 同一书籍、章节、页码只允许一个活动书签 |
| 异常处理 | 写入失败时回滚 UI 状态，并提示「书签更新失败」 |

### 阅读设置

| 设置项 | 范围 | 默认值 |
|---|---|---|
| 字体 | 系统字体 + 已导入字体 | 系统默认 |
| 字号 | 12-32pt | 24pt |
| 行距 | 1.0-2.5 | 1.4 |
| 段距 | 0.5-3.0 | 0.5 |
| 水平页边距 | 5%-20% | 7% |
| 垂直页边距 | 5%-20% | 2% |
| 主题 | 纯白、暖黄、薄荷、拿铁、深夜墨蓝、纯黑 OLED、自定义 | 暖黄 |
| 亮度 | 0%-100% | 跟随系统 |
| 护眼滤镜 | 关闭、暖黄、薄荷 | 关闭 |
| 翻页方式 | 仿真、覆盖、平移、上下、无动画 | 平移 |

设置变更后必须触发当前窗口重排版，但不得全书重排版。重排版期间保留当前页语义位置，优先保证用户仍停留在同一段落附近。

### 页面状态

| 状态 | UI 要求 |
|---|---|
| 首次加载 | 显示正文骨架屏，禁止翻页手势 |
| 正常阅读 | 工具栏隐藏，正文区域最大化 |
| 工具栏展开 | 顶部/底部控件出现，正文不跳动 |
| 加载下一页 | 当前页保持可读，目标页加载完成后再翻页 |
| 目录跳转中 | 显示轻量加载状态，允许取消返回当前页 |
| 空内容 | 提示「该章节暂无可阅读内容」，允许返回目录 |
| 解析失败 | 提示「文件解析失败」，提供返回书架 |
| 文件丢失 | 提示「文件已被移动或删除」，提供删除书架记录 |
| 内存警告 | 收缩缓存窗口到当前页 ±1 页，正文保持可读 |
